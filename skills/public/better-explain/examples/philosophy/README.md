# 哲学领域示例库

每个示例都遵循"思想实验 → 多派立场 → 开放问题"的哲学叙事逻辑。
注意：哲学问题通常没有收敛的答案，讲解的目标是呈现分叉、说明各派核心论证，而不是选边站。

## 示例：意识的难问题（The Hard Problem of Consciousness）

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 哲学僵尸思想实验、四派立场完整分叉、明确标注"开放问题"

**阶段1：问题定义**
> "你看到红色，大脑里有光感受器被激活，信号传到视觉皮层，神经元放电。这些我们都能解释。但是——'红色看起来是红色的感觉'是什么？这个主观体验（qualia）是从哪来的？为什么有'感觉什么'这回事，而不是只有信息处理？"

**阶段2：思想实验**
> "设想一个'哲学僵尸'（philosophical zombie）：外表行为和你完全一样，对'红色'做出同样反应，但内部没有任何主观体验——黑暗里只有信息处理，没有任何感觉。这在逻辑上似乎是可能的。如果可能，说明意识不能完全被物理解释；如果不可能，你需要说明为什么。"

**阶段3：认知分叉（四派立场）**
> - **物理主义/取消主义**（丹尼特）：意识的'难问题'是假问题，主观体验是我们用错误框架描述信息处理的结果。解决了功能问题，就解决了所有问题。
> - **泛心论**（查尔莫斯倾向）：意识是基本属性，就像质量或电荷一样，不能被还原。意识可能是物理世界的基本成分。
> - **高阶理论**：意识是对自身心理状态的表征——只有当系统能'思考自己在思考'，才有意识。
> - **整合信息论（IIT）**（托诺尼）：意识等于信息整合量（Φ）。Φ高的系统更有意识。这给出了可测量的预测，但被批评说大网格比人脑Φ更高。

**阶段4：开放性**
> "这个问题到今天没有收敛。争议的核心不只是答案，而是：我们甚至不确定什么样的答案会让人满意。这是哲学里极少数几个可能永远无法用科学方法解决的问题之一。"

## 示例：自由意志

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 盖奇事故（具体案例引入）、四派立场分叉、Libet实验的正确解读

**阶段1：问题定义**
> "你'决定'喝咖啡的那一刻，大脑神经元已经按照物理定律在活动。如果宇宙是决定论的，你的每个决定都是已经注定的结果——'你'做了什么决定？"

**阶段2：思想实验（盖奇事故）**
> "盖奇（Phineas Gage）事故：一根铁棒穿过他的额叶，他活下来了，智力基本完好，但人格彻底改变——从温和变得冲动粗鲁。他的'决定'方式改变了，是因为大脑硬件变了。如果大脑决定行为，'我'决定什么？"

**阶段3：认知分叉（四派立场）**
> - **硬决定论**：自由意志是幻觉。所有选择都是物理因果链的结果。
> - **相容论**（最主流）：自由意志和决定论不矛盾——自由意志是指'按照自己的欲望和理性行动，而不受外部强制'，不需要逃出物理因果。
> - **利伯塔里安自由意志**（不是政治意义）：存在真正的非决定性，可能来自量子随机性或心理因果。
> - **取消主义**（斯特劳森）：道德责任的概念需要彻底重建，因为没有人能从头选择自己的性格。

**阶段4：现实复杂性（Libet实验）**
> "Libet实验：让人随意决定什么时候弯手指，EEG显示大脑'准备电位'在'意识到决定'之前400-550毫秒就开始了。很多人解读为'意识在行动之后才出现'。但后来的研究发现这个解读过于简化——准备电位不等于不可撤回的决定，意识可能在'否决'这个决定上仍有作用。"

## 示例：知识论 — 我们怎么知道我们知道的东西？

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 笛卡尔恶魔思想实验、理性主义/经验主义/康德/实用主义四派分叉、二维概念空间表征层

**阶段1：问题定义**
> "你怎么知道外部世界是真实的？也许你只是缸中之脑，所有经验都是计算机模拟的。你能证明不是吗？"

**阶段2：思想实验（笛卡尔恶魔）**
> "笛卡尔的恶魔：假设有个全能的恶魔专门欺骗你，让你的所有感觉都是假的。你能找到任何无法被欺骗的东西吗？笛卡尔说：'我思故我在'——就算被欺骗，至少'正在被欺骗的那个思考者'是真实的。"

**阶段3：认知分叉（四派立场）**
> - **理性主义**（笛卡尔、莱布尼茨）：真正的知识来自理性，不来自感官。感官会欺骗我们，但逻辑数学不会。
> - **经验主义**（洛克、休谟）：所有知识都来自感官经验。没有天生的观念，大脑出生时是白板。
> - **康德的综合**：两者都对。有些知识是先验的（空间、时间、因果性是我们加在经验上的框架），有些是后验的（具体事实来自经验）。
> - **实用主义**（詹姆斯、杜威）：真理是有用的信念。不问'是否符合实在'，只问'是否有助于解决问题'。

**表征层（二维概念空间）**
> "知识论的核心张力可以用一个二维图来理解：一轴是'来源'（理性/经验），一轴是'确定性'（必然/偶然）。不同学派在这个空间里占据不同位置，争论的本质是：哪个位置能让人安心地说'我知道'。"
