# SVG Icon Library

This directory provides **640 high-quality SVG icons** that can be directly embedded into SVG files generated by PPT Master.

## Library

| Library | Style | Count | viewBox | Prefix |
|---------|-------|-------|---------|--------|
| `chunk` | fill · straight-line geometry (sharp corners, rectilinear) | 640 | `0 0 16 16` | `chunk/` |

---

## Usage

Use placeholder syntax **during SVG generation**:

```xml
<use data-icon="chunk/home" x="100" y="200" width="48" height="48" fill="#0076A8"/>
```

**Attributes**:
- `data-icon` — `chunk/<icon-name>` (filename without `.svg`)
- `x`, `y` — Position
- `width`, `height` — Size (recommend 32–48px for legibility)
- `fill` — Color

`finalize_svg.py` auto-embeds all placeholders during post-processing. To run manually:

```bash
python3 scripts/svg_finalize/embed_icons.py svg_output/*.svg
```

---

## Searching for Icons

Use `ls | grep` — zero token cost:

```bash
ls skills/ppt-master/templates/icons/chunk/ | grep home
```

---

## Style

`chunk` icons use **fill** style with straight-line geometry (M/L/H/V/Z only) — sharp, precise angles that are structured and highly legible. Well suited for academic, technical, and research presentations.

If the library doesn't have an exact icon, find the closest available alternative within `chunk`.
