## 示例：自然选择

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 认知分叉（达尔文 vs 拉马克）、表征层（适应度景观）

**阶段1：问题定义**
> "为什么同一物种的个体之间会有差异？差异有什么用？"

**阶段2：直觉设计**
> "想象一群灰色和白色的飞蛾落在树干上。工业革命前，树皮是浅色的，白色飞蛾更难被鸟发现。那白色的就活下来更多，留下更多后代。你注意到了什么？根本没有'意志'在决定颜色，只是'活下来的那些'恰好更可能把基因传下去。这就是自然选择的核心：不是主动进化，而是被动筛选。"

**阶段3：认知分叉**
> "当时达尔文和拉马克的观点在竞争。拉马克认为：长颈鹿努力伸脖子，获得的性状传给后代（获得性遗传）。达尔文认为：本来就存在脖子长短不同的个体，脖子长的活得更好，留下更多后代。两条路都有直觉基础，但实验最终支持了达尔文——获得性遗传没有分子机制支撑。"

**表征层**
> "自然选择可以理解为在'适应度景观'（fitness landscape）上的爬山算法——但爬的是整个种群，不是单个个体。山峰是高适应度，山谷是低适应度，进化就是种群在这个景观上的漂移和爬升。但这个景观本身也在随着环境变化——山在移动。"

**阶段4：现实复杂性**
> "自然选择不是万能的。遗传漂变（纯随机性）、性选择（孔雀尾巴明明降低生存率但被保留）、基因多效性（一个基因影响多个性状）都会让进化走出不'最优'的路。现实的进化是局部最优，不是全局最优。"

## 示例：CRISPR-Cas9

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 问题引入（细菌为什么没灭绝）、表征层（通缉令库+机器人+导航）、设计空间（Cas9/12/13/碱基编辑器）

**阶段1：问题定义**
> "细菌没有免疫系统，但病毒一直在攻击它们。几十亿年了，细菌为什么没灭绝？"

**阶段2：直觉设计**
> "细菌用了一个聪明的办法——它们把曾经入侵过的病毒DNA片段，像'通缉令'一样存档在自己基因组里。下次同样的病毒来了，就拿这张通缉令去比对，找到就剪断它。这就是CRISPR：细菌的'免疫记忆'。"

**表征层**
> "整个系统的结构是：CRISPR序列是存档的通缉令库；Cas9蛋白是带剪刀的分子机器人；guide RNA是给机器人指路的导航——告诉它去找哪段序列。三者组合起来，就是一个可编程的基因组编辑器。"

**阶段3：设计空间**
> "Cas9只是家族里的一个成员。Cas12更精准但效率低；Cas13专门切RNA不切DNA；碱基编辑器（base editor）可以单碱基修改而不切断双链。还有先导编辑（prime editing），相当于'搜索-替换'而不是剪切。每个工具在精准性、效率、脱靶率上各有取舍。"

**阶段4：现实复杂性**
> "在活体里，最大的挑战不是剪切，而是递送——怎么把Cas9送进目标细胞？腺相关病毒（AAV）太小装不下Cas9；脂质纳米颗粒（LNP）对肝脏效率高但难靶向其他组织。脱靶编辑的安全性还在评估中。这是从工具到疗法之间最难跨越的鸿沟。"

## 示例：细胞凋亡（Apoptosis）

**适用档位：** 🟢 入门 / 🟡 从业者
**核心技巧：** 动机引入（为什么细胞要主动自杀）、对比类比（凋亡 vs 坏死）、分叉（内源性 vs 外源性通路）

**阶段1：问题定义**
> "人体每天有500亿到700亿个细胞死亡。但这些细胞不是被'杀死'的，它们是自杀的。为什么细胞要主动自杀？"

**阶段2：直觉设计**
> "想象你在修剪盆栽——你需要剪掉一些枝条，整棵树才能长得更好。胚胎发育时，手指之间的蹼状组织需要被消除，才能形成独立的手指。癌变的细胞需要被清除，才不会失控增殖。细胞凋亡就是身体内置的'精准拆迁'机制。"

**表征层**
> "凋亡是一个高度有序的程序：细胞膜起泡（blebbing）→ 染色质浓缩 → DNA断裂成规则片段 → 细胞碎裂成'凋亡小体' → 被吞噬细胞清除，不引发炎症。与坏死（necrosis）的区别是：坏死是失控的，会释放内容物引发炎症；凋亡是有序的，像被打包好的垃圾。"

**阶段3：认知分叉**
> "凋亡有两条主要通路：内源性通路（线粒体通路，由细胞内部压力触发，如DNA损伤）和外源性通路（死亡受体通路，由外部信号触发，如免疫细胞发出的指令）。两条路最终汇聚到效应caspase，但触发机制和调控完全不同。"

**阶段4：现实复杂性**
> "凋亡失调是很多疾病的核心。癌细胞学会了逃避凋亡（上调Bcl-2，让线粒体通路失灵）；神经退行性疾病里神经元过度凋亡；自身免疫病里免疫细胞的凋亡清除失效。靶向凋亡通路是很多药物开发的核心方向——但调的是一把双刃剑。"
